# Synculariti - Expense Tracker (V2)

A minimalist, high-performance financial management platform designed for multi-user households.

🚀 **Live App**: [https://expense-tracker-v2-ten.vercel.app/](https://expense-tracker-v2-ten.vercel.app/)

---

## 🌟 The Core Value
**Multi-User Determinism**: V2 ensures that every household member sees the exact same data, financial insights, and budget state regardless of device.

## 🛠️ Tech Stack
- **Frontend/Backend**: [Next.js 14](https://nextjs.org/) (App Router)
- **Relational Data**: [Supabase](https://supabase.com/) (PostgreSQL)
- **Graph Analytics**: [Neo4j Aura](https://neo4j.com/)
- **Intelligence**: [Groq API](https://groq.com/) (Llama 3.3 70B)
- **Styling**: Premium Vanilla CSS (Bento-Grid / Glassmorphism)

## 📱 PWA (2026 Standards)
Synculariti is designed as a Progressive Web App.
- **Mobile-First**: Identity-driven header, safe-area inset support, and orientation locking.
- **Offline Resilience**: Exponential backoff and state retention for flaky mobile connections.
- **Atomic Operations**: Financial integrity via Postgres RPC transactions.

## 🧠 Intelligence Strategy
We employ a **Cloud-TTL (24h)** strategy to provide expensive AI insights across the household while minimizing API costs and latency.
